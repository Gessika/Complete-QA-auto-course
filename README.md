# QA Automation final project

This project is a collection of **UI** and **Database** and **API** automated tests, created as part of my QA Automation learning path.

Tests are written using **Python**, **PyTest**, **Selenium WebDriver** and **SQLLite3**.

## Sections
- **├── config/** — configuration files (test data, environment, etc.)
- **├── modules/** — Python modules with main functions or logic
- **├── tests/** — Automated test cases (pytest)
- **├── conftest.py/** - Pytest configuration and fixtures
- **├── pytest.ini/** - Pytest settings
- **├── become_qa_auto.db/** - SQLite database example
- **├── README.md/** - Project description
- **├── requirements.txt/** - Project dependencies
- **├── .gitignore.py/** - exclude unnecessary files and folders (files that are automatically generated by Python, the operating system, or IDEs)

## How to Run Tests

Follow these steps to run the automated tests in this project.

### 1️⃣ Clone the repository
```
git clone https://github.com/Gessika/Complete-QA-auto-course.git
cd Complete-QA-auto-course
```
### 2️⃣ Install dependencies:
```
   pip install -r requirements.txt
```
### 3️⃣ Run all tests:
``` 
pytest -v
```
### 4️⃣ Run tests by type:
   * UI tests only:
```
   pytest -s -m ui
```
  * Database tests only:
```
  pytest -s -m database
```
  * API tests only:
```
  pytest -s -m api
```
